# GreenStatesBD Backend

A scalable backend system that powers the GreenStatesBD real estate bidding platform. It handles property management, bidding workflows, user roles, moderation, and media processing with a focus on security and performance.

## Tech Stacks

- Node.js with Express.js and TypeScript
- PostgreSQL managed via Prisma ORM (NeonDB)
- Firebase Admin SDK for authentication and authorization
- Cloudinary for media storage and optimization
- Groq API for AI integration
- Zod for schema validation

---

## Features

- Secure authentication and role-based authorization using Firebase Admin.
- Property listing system with full CRUD functionality.
- Blog management system with AI integration support.
- Report and moderation system for handling user-generated content.
- Media upload pipeline with Cloudinary optimization.
- Structured REST API for seamless frontend integration.
- Scalable and modular architecture for long-term maintainability.
